# Gerador de Protocolo

Página estática para geração local de protocolos de atendimento.

## Como publicar na Hostinger

Envie os arquivos abaixo para a pasta pública do site, normalmente `public_html`:

- `index.html`
- `protocolo.css`
- `protocolo.js`

Não há dependência de backend, login, banco de dados ou autenticação. O protocolo é gerado no navegador do usuário.

## Formato do protocolo

O padrão atual é:

```text
DDMMAAHHMMSSmmm
```
